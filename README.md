# BaseProjectDemo
this is BaseProject demo

## Version
这是第二版，第一个版本在OsChina上面。

## 描述
这个项目是作为一个项目基础lib的，用来快速进行Android开发。
这个项目集成类似RxJava2、Retrofit2、Butterknife等一些常用工具类。
